# webDriver 破解滑动验证码

下载 firefox
下载 [geckodriver](https://github.com/mozilla/geckodriver/releases) 解压，windows放到 python 的目录下，mac放到 /usr/local/bin目录

然后 `npm install``node index.js` 就可以实现啦（因为图形的算法问题，准确率不高，但实现效果已经足够了~）
